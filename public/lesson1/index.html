Introduction to Html


What is HTML

Html is a Markup Language (HTML = Hyper Text Markup Language) used to write web pages. It tells the browser what kind of content is on the page and describes how the browser should display this content. Every website online is written in HTML, even this one.

HTML TAGS

Html tags are the “elements” of Html that tell the browser how to display certain parts of the page


       <h1> This is a Heading</h1>

So for example, you can tell the browser that the text “This is a Heading” is a heading by wrapping it around the heading tags <h1> and </h1>

<h1> is the “Opening Tag” that you must put at the beginning of the text you want to describe, while </h1> is the “Closing Tag” that you put at the end.

Other examples of html tags include <h2>...</h2> for sub-headings (you can use h1 to h6 to specify headings of differing importance), <p>...</p> for paragraphs and <a>...</a> for links. I'll introduce you to more HTML tags pretty soon.

As you can probably already tell, all tags are structured this way (<tag>...</tag>) except for special tags like <br /> which are self closing tags. <br /> is used to insert a single line break in your content. You'll learn more about self closing tags pretty soon.

Attributes

Atttributes give the browser more information about the html elements you are using to describe the content of your page. The attributes are positioned in the opening tag of your html element


    <tag attribute=“value”>...<tag>

For example, the <img /> tag is a self-closing html tag used to tell the browser to display an image. But what image should the browser display? Where is it located?

You can use the (src=“” ) attribute to tell the browser where to find the image you want it to display. In the quotation marks, you put a link to the image you want the browser to display.


    e.g. <img src=“http://eofdreams.com/data_images/dreams/cat/cat-07.jpg” />

The <a></a> tag is another example of a tag that needs an attribute to work. The <a></a> is used to insert a hyperlink into the document but how does the browser know what to link to?

You can use the (href=“” ) attribute to tell the browser what to link to.


    e.g. <a href=“http://www.google.com” >this is a link</a>

HTML Web Page Structure

Every web page on the internet starts like this:


    <!DOCTYPE html>
     <html>

        <head>
        </head>

        <body>
        </body>

     </html>

It must have all the html tags above to be valid web page.

<!DOCTYPE html> - this tells the web browser what version of html your web-page (I.e, an HTML document) is written in

<html></html> - acts as a container for the html document, and all other html elements must be inside this one.

<head></head> - contains all the tags that give the browser more information about the web-page and how it should behave. For example, it contains the tags, which gives the browser a Title for the web-page that will appear on the title-bar (or tab) in the browser. It also contains meta tags (which further describe the contents of the page and how it should behave), script files, style sheets (css) and links to external stylesheets and script files.

<body></body> - This is the part of the document that contains all the “content” of the web page.
